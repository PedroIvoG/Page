Visão geral

Este projeto consiste em um dashboard interativo de vendas desenvolvido no Power BI com integração direta a uma base de dados SQL Server. O objetivo foi centralizar os dados de vendas de múltiplas filiais em um único painel, permitindo análise em diferentes granularidades.

O problema

A empresa não possuía visibilidade centralizada de suas vendas. Os dados estavam distribuídos em planilhas Excel por filial, dificultando o acompanhamento das metas e a identificação de tendências sazonais.

Solução desenvolvida

Desenvolvi um pipeline de dados que extraía, transformava e carregava os dados consolidados para o Power BI. O dashboard incluiu:

  • Acompanhamento de meta mensal com gauge e indicadores de variação
  • Análise de sazonalidade com visualização por mês/trimestre/ano
  • Mapa de calor por região com drill-through para filial
  • Ranking de produtos por receita, ticket médio e margem
  • Filtros dinâmicos com slicers sincronizados

Exemplo de medida DAX

-- Variação vs Meta (YTD) Var_Meta_YTD = VAR VendasYTD = CALCULATE( SUM(Vendas[Valor]), DATESYTD(Calendario[Data]) ) VAR MetaYTD = CALCULATE( SUM(Metas[ValorMeta]), DATESYTD(Calendario[Data]) ) RETURN DIVIDE(VendasYTD - MetaYTD, MetaYTD, 0)

Resultados

Com o painel em produção, a equipe de gestão passou a tomar decisões com base em dados atualizados diariamente, reduzindo em 70% o tempo gasto na consolidação manual de relatórios.
